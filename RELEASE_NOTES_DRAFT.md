# Release 草稿（Release Notes - Draft）

版本：v0.1.0（示例）
发布日期：2026-02-05

主要更新
- 初始发布：PictureView 的第一版二进制/源码发布，支持单图/缩略图/幻灯片浏览功能。
- 平台：基于 Qt，已在 Linux (Ubuntu) 上测试构建。

已知问题
- Windows 与 macOS 的构建脚本可能需要额外调整（参考 README 中的 CMake / qmake 指南）。
- 部分特性在不同桌面环境下表现差异，后续迭代将继续改进兼容性。

安装与下载
- 源码：通过 GitHub 仓库克隆并按 README 构建。
- 二进制（如果有）：附件或下载链接将放在此处。

安全说明
- 如发现安全漏洞，请按照 SECURITY.md 中的流程进行报告。
