# 安全报告（SECURITY.md）

如果你在使用 PictureView 时发现安全漏洞或敏感的安全问题，请通过 GitHub 的安全联系人（在仓库设置中配置）或私下渠道联系维护者并报告细节。请在报告中尽量包含复现步骤、受影响的版本和详细日志（如果适用），以便我们及时跟进。

安全联系人
- 维护者：berylok
- 首选联系方式：使用 GitHub 安全联系人（请在仓库设置中添加或检查）

漏洞披露流程
1. 请优先通过安全联系人或私有渠道提交漏洞报告（避免在公开 issue 中披露详细漏洞细节）。
2. 我们会在 7 天内确认收到并给出初步响应（如果需要更多时间，会告知预计的进度）。
3. 讨论与修复将优先在私下进行；修复后会发布包含修复说明的 release。
4. 若问题已修复并发布，我们会在公开渠道（Release Notes / CHANGELOG）中说明修复情况。

免责声明
- 感谢负责任的披露 — 我们会尽力在合理时间内处理并修复问题.
